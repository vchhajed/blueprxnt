# Blueprxnt Website

A modern, professional website built with clean HTML, CSS, and JavaScript.

## Design System

### Colors
- Primary Blue: `#0014cc`
- Secondary Blue: `#3372e3`
- Light Blue: `#86aaed`
- White: `#ffffff`
- Black: `#000000`

### Typography
- **Heading**: Poppins (Google Fonts)
- **Subheading**: Poppins
- **Body**: Inter (Google Fonts)
- **Title**: Daniel Black

## Features

- Responsive design that works on all devices
- Animated dot background pattern
- Smooth scroll navigation
- Interactive contact form with validation
- FAQ accordion
- Mobile-friendly hamburger menu
- Scroll animations and parallax effects
- Professional service showcase
- Team member profiles
- Statistics counter animation

## Pages

1. **Home** ([index.html](index.html)) - Landing page with hero section and overview
2. **About** ([about.html](about.html)) - Company story, values, team, and impact
3. **Services** ([services.html](services.html)) - Detailed service offerings and process
4. **Contact** ([contact.html](contact.html)) - Contact form, FAQ, and contact information

## File Structure

```
Blueprxnt/

├── contact.html        # Contact page
```

## Usage

Simply open `index.html` in a web browser to view the website. All pages are interconnected through the navigation menu.

## JavaScript Features

- Mobile menu toggle
- Smooth scrolling
- Scroll animations with Intersection Observer
- Contact form validation and submission
- FAQ accordion functionality
- Active navigation highlighting
- Statistics counter animation
- Parallax effects

## Browser Support

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Customization

To customize the website:

1. **Colors**: Edit CSS variables in `styles.css` (lines 2-8)
2. **Fonts**: Change Google Fonts import in HTML files
3. **Content**: Edit HTML files directly
4. **Animations**: Modify JavaScript in `script.js`
